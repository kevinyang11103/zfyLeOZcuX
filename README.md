# 学生档案管理系统 java613

## 项目概述

学生档案管理系统是基于 Spring Boot、MyBatis 和 MySQL 技术栈开发的一个平台。系统采用前后端分离的设计，前端使用 VUE 框架构建用户界面。

## 功能模块

### 核心功能

- 学生基本信息管理
- 课程学业信息管理
- 活动竞赛信息管理
- 健康体检信息管理
- 违纪信息管理
- 就业发展信息管理
- 档案管理
- 档案借阅流程控制

### 系统支撑

- 系统管理

## 系统角色

- 学生
- 教师管理员
- 系统管理员

## 技术栈

- 后端：Spring Boot、MyBatis
- 数据库：MySQL
- 前端：VUE

## 运行环境

- 操作系统：兼容性良好，可在主流操作系统上运行
- 服务器：支持 Java 运行的服务器
- 客户端：现代浏览器

## 部署步骤

1. 准备兼容的 Java 运行环境
2. 部署 MySQL 数据库，并执行初始化 SQL 脚本
3. 部署后端服务，配置数据库连接
4. 部署前端 VUE 应用，确保与后端服务交互正常
